# 3조 팀원 소개 페이지

Git 커밋 협업을 통한 우리팀 팀원 소개 페이지 입니다.

## 팀원 역할 안내

- **권지원** : 프론트엔드 개발자 - React, TypeScript 중심
- **김예은** : 백엔드 개발자 - Node.js, Python 중심
- **최길중** : 풀스텍 개발자 - React, Node.js 모두
- **류호정** : UI/UX 디자이너 - Figma, Adobe XD 중심
- **이재민** : 데이터 분석가 - Python, SQL, 머신러닝 중심

템플릿 활용하여 각자 역할에 맞는 소개 카드의 내용 수정
